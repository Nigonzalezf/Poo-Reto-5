# Poo-Reto-5
Para este reto se busca organizar clases mediante la creación de paquetes y modulos.

## Con un modulo _Shape_ unico
En este caso 

```
Shapes/
│   ├── __pyache__/
│   ├── __init__.py
│   ├── shape.py
└── main.py
```

## Modulos Individuales

Shapes/

│   ├── point.py
│   ├── line.py
